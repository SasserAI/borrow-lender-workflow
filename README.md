# borrow-lender-workflow
Private single-user workflow tool for my own Reddit activity related to r/Borrow. 

## Purpose
This project will help me review r/Borrow requests more efficiently using public Reddit data.

## What it will do
- Monitor new posts in r/Borrow
- Apply my saved filters to those posts
- Review public borrower activity, such as recent posts and comments
- Show relevant public activity in subreddits that I exclude from consideration
- Help me manage my own private loan records, reminders, and notes

## Scope
- Private single-user use
- Read-only
- No bulk messaging
- No spam automation
- No moderation actions

## Status
Prototype in development. This repository currently only contains project documentation.
